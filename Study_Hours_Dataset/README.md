Student Performance Analysis: Linear & Multiple Regression 

This project explores the relationship between study habits and academic success using Linear Regression and Multiple Linear Regression models. It analyzes how single or multiple variables (like study hours and other factors) influence student grades.

Project Structure
1-studyhours.csv: Dataset for Simple Linear Regression (Study Hours vs. Scores).

2-multiplegradesdataset.csv: Dataset for Multiple Linear Regression (multiple independent variables).

Study_Hours_Dataset.ipynb: Jupyter Notebook containing data cleaning, exploratory data analysis (EDA), and model implementation.

Tech Stack
Language: Python

Libraries: 
* Pandas & NumPy: For data manipulation and handling the datasets.

*Matplotlib & Seaborn: For visualizing the regression line and correlation heatmaps.

*Scikit-Learn: For training the models and extracting key metrics like Coefficients and Intercepts.

Key Objectives
Simple Linear Regression: To determine how much a single unit increase in study time affects the final grade.

Multiple Linear Regression: To analyze the combined impact of various factors on student performance.

Prediction: Building a model that can predict a student's grade based on their input data.

Summary of Concepts
Coefficient: The slope of the line, indicating the change in the score for every additional hour studied.

Intercept: The baseline score when the independent variables are zero.
